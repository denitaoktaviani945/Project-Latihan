CSS Flexbox

flex Box adalah model layout 1 dimensi yang dapat mengatur jarak dan pembelajara antar item dalam sebuah container. mengapa di sebut satu dimensi karena hanya dapat mengatur 1 dimensi pada saat tertentu antara baris atau kolom tidak bisa keduanya sekaligus

property pada container

  • display

    display : flex;
    membuat sebuah elemen parent menjadi flexbox dan membuat elem di dalam nya berprilaku flex juga

  • flex-direction

    Berfungsi mengatur arah dan urutan dari item di dalam container
    row = dari kiri kr kanan
    row-reverse = dari kanan ke kiri
    column = dari atas ke bawah
    column-reverse = dari bawah ke atas

  • flex-warp

    secara default semua items di dalam container akan berada pada satu baris meskipun ukuranya sudah tidak cukup, warp memungkinkan untuk memindahkan items ke baris di bawahya

    1. nowrap = maksa agar tidak ke bawah dan jika bertambah maka akan ke piggir
    2. wrap = dia akan berpindah ke bawah
    3. wrap-reverse = kebalikan dari warp dia bermulai dari bawah ke atas
